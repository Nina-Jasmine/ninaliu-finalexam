# Final Exam Preparation API

This is a RESTful API project for the Final Exam of COMP3033. It allows you to manage a collection of restaurants. The API supports CRUD operations, pagination, and basic authentication for creating and deleting resources. Swagger UI is included for easy API documentation and testing.
